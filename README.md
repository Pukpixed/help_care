# ระบบติดตามสุขภาพผู้สูงอายุและผู้ป่วยติดเตียง
(Elderly & Bedridden Health Monitoring System)

## 📌 รายละเอียดโครงการ
ระบบนี้ถูกพัฒนาขึ้นเพื่อใช้ในการติดตาม บันทึก และประเมินข้อมูลสุขภาพของผู้สูงอายุและผู้ป่วยติดเตียง โดยช่วยให้ผู้ดูแลสามารถตรวจสอบข้อมูลสุขภาพได้แบบเรียลไทม์ พร้อมระบบแจ้งเตือนเมื่อค่าผิดปกติ

---

## 🎯 วัตถุประสงค์
- บันทึกข้อมูลสุขภาพรายวัน
- ติดตามสัญญาณชีพ เช่น ความดันโลหิต ชีพจร อุณหภูมิ
- แจ้งเตือนเมื่อค่าผิดปกติ
- จัดเก็บประวัติสุขภาพย้อนหลัง
- เพิ่มความสะดวกให้ผู้ดูแล

---

## 👥 กลุ่มผู้ใช้งาน
- ผู้ดูแล (Caregiver)
- บุคลากรทางการแพทย์
- ผู้สูงอายุ / ผู้ป่วยติดเตียง (ผ่านผู้ดูแล)

---

## 🛠 เทคโนโลยีที่ใช้
- Frontend: Flutter / HTML / CSS / JavaScript
- Backend: Node.js / PHP
- Database: Firebase / MySQL
- Cloud Service: Firebase Cloud Messaging (แจ้งเตือน)

---

## 🗂 โครงสร้างระบบ
- ระบบสมัครสมาชิก / เข้าสู่ระบบ
- บันทึกข้อมูลสุขภาพ
- แสดงกราฟสถิติสุขภาพ
- ระบบแจ้งเตือน
- รายงานสรุปข้อมูล

---

## 📊 ตัวอย่างข้อมูลที่บันทึก
- ความดันโลหิต (Blood Pressure)
- ชีพจร (Pulse Rate)
- อุณหภูมิร่างกาย (Body Temperature)
- ระดับน้ำตาลในเลือด (Blood Sugar)
- การให้ยา

---

## 🔐 ความปลอดภัยของข้อมูล
- เข้ารหัสข้อมูลผู้ใช้งาน
- กำหนดสิทธิ์การเข้าถึงข้อมูล
- ระบบยืนยันตัวตนก่อนเข้าใช้งาน
