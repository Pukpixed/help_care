# ระบบติดตามสุขภาพผู้สูงอายุและผู้ป่วยติดเตียง
(Elderly & Bedridden Health Monitoring System)

## 📌 รายละเอียดโครงการ
ระบบนี้ถูกพัฒนาขึ้นเพื่อใช้ในการติดตาม บันทึก และประเมินข้อมูลสุขภาพของผู้สูงอายุและผู้ป่วยติดเตียง โดยช่วยให้ผู้ดูแลสามารถตรวจสอบข้อมูลสุขภาพได้แบบเรียลไทม์ พร้อมระบบแจ้งเตือนเมื่อค่าผิดปกติ

---

## 🎯 วัตถุประสงค์
- บันทึกข้อมูลสุขภาพรายวัน
- ติดตามสัญญาณชีพ เช่น ความดันโลหิต ชีพจร อุณหภูมิ
- แจ้งเตือนเมื่อค่าผิดปกติ
- จัดเก็บประวัติสุขภาพย้อนหลัง
- เพิ่มความสะดวกให้ผู้ดูแล

---

## 👥 กลุ่มผู้ใช้งาน
- ผู้ดูแล (Caregiver)
- บุคลากรทางการแพทย์
- ผู้สูงอายุ / ผู้ป่วยติดเตียง (ผ่านผู้ดูแล)

---

## ✨ ฟีเจอร์หลัก (Core Features)
- **โปรไฟล์ผู้ป่วย/ผู้สูงอายุ**
  - เพิ่ม/แก้ไขข้อมูลผู้ป่วย เช่น ชื่อ อายุ โรคประจำตัว ผู้ดูแลหลัก
- **ตารางการให้ยา + แจ้งเตือน**
  - ตั้งเวลาเตือนแบบรายวัน/ตามช่วงเวลา  
  - รองรับเสียงเตือน/สั่น (Local Notifications)
- **บันทึกกิจวัตรประจำวัน**
  - เช่น วัดความดัน น้ำตาล อาหาร การขับถ่าย การทำแผล/กายภาพ
- **สรุปผลรายวัน/รายเดือน/รายปี**
  - ดูภาพรวมพฤติกรรมและแนวโน้มสุขภาพ
- **ประวัติสุขภาพย้อนหลัง**
  - แสดงข้อมูลย้อนหลังจากบันทึกกิจวัตรในรูปแบบที่ดูง่าย
- **นัดหมาย**
  - บันทึกวันนัด หมอ/โรงพยาบาล พร้อมแจ้งเตือนล่วงหน้า
- **เอกสารสุขภาพ**
  - เก็บรูป/ไฟล์เอกสาร เช่น ใบนัด ผลตรวจ ใบยา
- **SOS/Buddy**
  - แจ้งเหตุฉุกเฉินไปยังผู้ดูแล/คนในครอบครัวได้ทันที
- **ข่าวสารการดูแล**
  - ลิงก์ไปยังแหล่งข้อมูลด้านสุขภาพ เช่น กรมอนามัย/สสส/โรงพยาบาล

## 🚀 การติดตั้งระบบ
## 🚀 การติดตั้งระบบ (สำหรับผู้ใช้งานที่มีไฟล์ .apk แล้ว)

### 1) เตรียมเครื่อง
- โทรศัพท์ Android
- อินเทอร์เน็ต (แนะนำเพื่อเข้าสู่ระบบ/ซิงก์ข้อมูล)
- ไฟล์แอปนามสกุล `.apk` (เช่น `helpcare.apk`)

---

### 2) อนุญาตให้ติดตั้งแอปจากแหล่งที่ไม่รู้จัก (Unknown Sources)
> Android จะบล็อกการติดตั้งไฟล์ APK ครั้งแรก ต้องอนุญาตก่อน

1. กดเปิดไฟล์ `.apk` ที่ดาวน์โหลดไว้ (จาก Files / Download)
2. ระบบจะแสดงหน้าต่างแจ้งเตือน → กด **Settings / การตั้งค่า**
3. เปิด **อนุญาตจากแหล่งนี้ (Allow from this source)**  
   หรือ **ติดตั้งแอปที่ไม่รู้จัก (Install unknown apps)**

---

### 3) ติดตั้งแอป (Install)
1. เปิดไฟล์ `.apk`
2. กด **Install / ติดตั้ง**
3. รอจนเสร็จ → กด **Open / เปิด**

---

### 4) ตั้งค่าที่จำเป็นหลังติดตั้ง (สำคัญมาก)
เพื่อให้ “แจ้งเตือนทานยา” ทำงานได้ครบ แนะนำเปิดค่าต่อไปนี้

#### 4.1 อนุญาตการแจ้งเตือน (Notifications)
- ไปที่ **Settings → Apps → HelpCare → Notifications**
- เปิด **Allow notifications / อนุญาตการแจ้งเตือน**

#### 4.2 อนุญาตทำงานเบื้องหลัง (Background)
> มือถือบางรุ่นจะปิดแอปอัตโนมัติเพื่อประหยัดแบต ทำให้แจ้งเตือนไม่เด้ง

- ไปที่ **Settings → Apps → HelpCare → Battery**
- เลือก **Unrestricted / ไม่จำกัด** (หรืออนุญาตทำงานเบื้องหลัง)

#### 4.3 ปิด Battery Optimization (ถ้ามีเมนูนี้)
- ไปที่ **Battery → Battery Optimization**
- หา **HelpCare** → ตั้งเป็น **Don’t optimize**

---

### 5) วิธีอัปเดตแอป (Install ทับเวอร์ชันเดิม)
หากมีไฟล์ APK เวอร์ชันใหม่:
1. ดาวน์โหลดไฟล์ `.apk` ใหม่
2. กดติดตั้งทับได้เลย (ระบบจะอัปเดตแทนการติดตั้งใหม่)
> ถ้าติดตั้งไม่ได้ ให้ถอนการติดตั้งเวอร์ชันเดิมก่อน แล้วติดตั้งใหม่
